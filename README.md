# Система анализа земельных участков с применением методов искусственного интеллекта

Выпускная квалификационная работа

Тема:
**«Применение методов искусственного интеллекта для распознавания и обработки документов в системе 1С»**

## Описание проекта

Проект представляет собой информационную систему для автоматизации анализа земельных участков на основе документов и спутниковых данных.

Система позволяет:

- получать сведения о земельных участках по кадастровому номеру;
- отображать участки на карте;
- получать спутниковые снимки Sentinel-2;
- классифицировать состояние земельных участков с использованием модели машинного обучения;
- извлекать текст из документов (OCR);
- выполнять интеллектуальный анализ документов с помощью LLM;
- интегрировать результаты в интерфейс 1С.
